# Pokémon Team Showcase

A public showcase of competitive Pokémon teams, with a 3D presentation layer.

- **Anyone can view** the teams (public, read-only).
- **Only the owner can add, edit, or delete** teams (protected by a single admin token).
- Teams are managed using the **Pokémon Showdown copy-paste text format**.

## Tech stack

**Frontend:** React + Vite (SPA) · React Three Fiber (3D) · TanStack Query · Tailwind CSS · TypeScript
**Backend:** Node + Express · Prisma · SQLite (→ Postgres later) · TypeScript
**Pokémon data:** the [`@pkmn`](https://github.com/pkmn) packages (community-maintained, open-source — not Nintendo/Game Freak assets)

🚧 Early development. Built one layer at a time — see the build roadmap in [`CLAUDE.md`](./CLAUDE.md).
